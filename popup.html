How to Use:

  • When you have opened any google app(gmail, calendar, keep etc).Press alt + num to switch to a particular account number
  • For example for switching between Gmail accounts, try pressing alt + 1, alt + 2 etc
  • Almost all google apps are supported. Send a request if any app is not covered.
  • For support or any other issue, contact: email
  • Source code for the extension is open sourced at: Github
